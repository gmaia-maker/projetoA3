# Sobre o projeto

O Projeto A3 consiste no desenvolvimento de um sistema de gerenciamento para supermercado, implementado na linguagem Java e executado em ambiente de console. O objetivo principal é simular as operações básicas realizadas em um estabelecimento comercial, proporcionando uma experiência prática na aplicação dos conceitos de programação orientada a objetos, manipulação de arquivos e estruturas de dados.
O sistema permite o cadastro, consulta, alteração e exclusão de clientes e produtos, armazenando informações essenciais para o funcionamento do supermercado. Os produtos podem ser selecionados durante o processo de compra, possibilitando a montagem de um carrinho virtual de compras.
Além disso, o projeto conta com um sistema funcional de caixa, responsável por registrar as vendas realizadas, calcular automaticamente os valores totais da compra, controlar a quantidade de itens adquiridos e emitir um resumo da transação. O sistema também realiza a atualização do estoque após cada venda, garantindo maior controle sobre os produtos disponíveis.
